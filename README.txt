A simple wrapper for libtag's C, abstract API

Tested on:
	   -Ubuntu 13.04, x86 64-bit
	   -Clozure 1.9, 64-bit (note: you need to follow directions and create the CDB interface lib for CCL (see 13.7 in CCL manual))
	   -libtag version 1.8 installed in /usr/local

Make your life simple---install quicklisp to get all the dependencies loaded!
